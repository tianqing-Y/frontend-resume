# Changelog

All notable changes to this skill will be documented here.

## [2.0.0] - 2024-01

### Added
- 三阶段 10 步结构化执行流程（Phase 1-3）
- Step 2：AI 使用经历采集模块
- Step 3：转型意向采集模块
- Step 7.3：AI 经历简历描述模板（三种场景）
- Step 7.4：转型意向简历策略（全栈/后端/产品）
- Step 8.5：AI 相关面试题（技术类 + 认知类）
- F 类隐性亮点：「产品说要能自己改」（配置化/运营驱动需求）
- 工作年限分层策略（1-3年 / 3-5年 / 5年+）
- 流程总览图（ASCII 结构图）
- 每步进度标注规范 `[Phase X · Step Y 完成]`

### Changed
- Step 编号统一为两级结构（Phase X · Step Y），消除 Step 2.5 等异常编号
- 隐性亮点从 A-E 类扩展为 A-F 类
- 简历描述输出格式新增「来源」标注字段
- 面试题章节细化编号（8.1-8.5）
- 最终汇总输出新增 AI 模块和转型模块

### Fixed
- 修复优先级表格中对「来源」字段描述不一致的问题
- 统一【必做】标注为 ⛔ 强制符号，提升视觉优先级
